Tool: DICOM Metadata

DICOM files contain associated information (metadata) that can be displayed in the DICOM Metadata viewer and the related DICOM Browser tool.

DICOM Metadata shows a full list of DICOM attributes. This list can be shown by choosing one or more series in the Model Panel and clicking the Info button. It can also be shown by using the context menu in the DICOM Browser tool. If shown for patient or study, the attributes of all of the corresponding series will be included.

Columns include tags, attribute names, and attribute values. Doubleclicking a row opens an explanation (if available) in the Help Viewer for that specific attribute at dicomlookup.com.
